#  EFREI Covoiturage

Plateforme de covoiturage dédiée à la communauté étudiante d'EFREI Paris.

##  Lancer le projet

### Prérequis
- Node.js (v18+)
- npm

### 1. Lancer le serveur (back-end)

```bash
cd server
npm install
npm run dev
```
Le serveur tourne sur **http://localhost:3000**

### 2. Lancer le client (front-end)

```bash
cd client
npm install
npm run dev
```
L'application tourne sur **http://localhost:5173**

##  Technologies

## 🛠️ Technologies

- **Front-end** : Vue.js 3, Vue Router, Axios
- **Back-end** : Node.js, Express.js
- **Base de données** : SQLite
- **Sécurité** : bcryptjs (hashage mots de passe), JWT (authentification)
